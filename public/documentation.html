iFlix API // Developers // v1.0

Documentation

iFlix is a server-side web application that allow users to access information about different movies. Users will also be able to sign up, update personal information, and create lists of favorites!

Resource URL

https://i-flix.herokuapp.com/

API Reference

HTTP Method URL DESCRIPTION
GET  /movies Return a list of all movies to the user
GET  /movies/:title Get data (description, genre, director, Image url) about a single movie by title
GET  /movies/genre/:genre Get data about a genre
GET  /movies/director/:director Get data about a director (bio, birth year, death year)
POST  /users Add a new user
PUT  /users/:username Update users info (username, password, email, date of birth)
POST  /users/:username/favorites/:movieId Add a movie to users list of favorites
DELETE  /users/:username/favorites/:movieId Remove a movie from users list of favorites
DELETE  /users/:username Remove a user
